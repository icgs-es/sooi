Política de privacidad

Última actualización: mayo de 2026.

1. Responsable

El responsable del tratamiento de los datos asociados a SOOI es {{ system_settings.company_name|default:"ICGS Gestión & Servicios S.L." }}. Para consultas relacionadas con privacidad puedes escribir a info@sooi.io.

2. Datos tratados

SOOI puede tratar datos de registro, contacto, solicitudes de demo, datos operativos introducidos por el usuario, búsquedas, captaciones, tareas, alertas y notas internas asociadas al seguimiento inmobiliario.

3. Finalidad

Los datos se utilizan para prestar el servicio, gestionar el acceso de usuarios, responder solicitudes de demo, operar el sistema de oportunidades inmobiliarias y mejorar la seguridad y trazabilidad del producto.

4. Datos de inmuebles y contactos

SOOI está diseñado para minimizar la información personal almacenada. Se recomienda registrar solo la información estrictamente necesaria para el seguimiento operativo y evitar copiar datos personales innecesarios desde portales externos.

5. Conservación

Los datos se conservarán mientras exista una relación activa con el usuario o mientras sean necesarios para la prestación del servicio, salvo solicitud de eliminación o bloqueo cuando corresponda.

6. Seguridad

SOOI aplica medidas técnicas razonables como autenticación, control de acceso, aislamiento por usuario, copias de seguridad y gestión controlada de credenciales.

7. Derechos

Puedes solicitar acceso, rectificación, eliminación o limitación del tratamiento escribiendo a info@sooi.io.

8. Cambios

Esta política podrá actualizarse para reflejar cambios funcionales, comerciales o normativos del servicio.
